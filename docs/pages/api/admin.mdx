# Admin API

All admin APIs require authentication. Include `Authorization: Bearer <admin-token>` on every request.

<Callout>
  Get an admin token by calling `POST /auth/sign-in/email` with admin credentials.
</Callout>

---

## Auth

### Sign In
`POST /auth/sign-in/email`

```json
{ "email": "admin@workit.co.ke", "password": "password" }
```

### Sign Out
`POST /auth/sign-out`

### Get Session
`GET /auth/session`

## Analytics

All analytics endpoints require the `analytics.view` permission.

| Endpoint | Description |
|----------|-------------|
| `GET /analytics/dashboard-overview` | Full dashboard data (summary, charts, recent orders) |
| `GET /analytics/weekly-stats` | Orders, revenue, customers, products counts |
| `GET /analytics/weekly-chart` | Revenue trend data for charts |
| `GET /analytics/sales-stats` | Total sales with growth percentage |
| `GET /analytics/recent-orders` | Latest orders (default: 5) |

## Catalog — Products

Requires `catalog.manage` permission.

| Method | Endpoint | Description |
|--------|----------|-------------|
| `GET` | `/catalog/products/admin` | List products with filters |
| `POST` | `/catalog/products/admin` | Create product |
| `GET` | `/catalog/products/admin/search?q=` | Search products |
| `GET` | `/catalog/products/admin/:id` | Get product |
| `PUT` | `/catalog/products/admin/:id` | Update product |
| `PATCH` | `/catalog/products/admin/:id` | Partial update |
| `DELETE` | `/catalog/products/admin/:id` | Delete product |
| `POST` | `/catalog/products/admin/bulk-delete` | Bulk delete `{ ids: [] }` |
| `GET` | `/catalog/products/admin/template` | Download CSV import template |
| `GET` | `/catalog/products/admin/export` | Export all products as CSV |
| `POST` | `/catalog/products/admin/import` | Import products from CSV data |
| `POST` | `/catalog/products/admin/search/reindex` | Trigger Algolia reindex |

**Create/Update product body:**
```json
{
  "name": "Product Name",
  "slug": "product-name",
  "sku": "SKU-001",
  "description": "Description",
  "originalPrice": 9999,
  "salePrice": 7999,
  "stockOnHand": 50,
  "enabled": true,
  "condition": "NEW",
  "brandId": "uuid",
  "collections": ["uuid1", "uuid2"],
  "assetIds": ["uuid1", "uuid2"]
}
```
